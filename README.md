# 🧠 Sistema de Seguimiento Psicológico Anónimo

Aplicación web para el seguimiento del bienestar emocional de pacientes, garantizando anonimato mediante seudónimos 👤🔒 y apoyando la toma de decisiones clínicas con datos y estadísticas 📊.

El sistema está compuesto por Frontend (cliente web) y Backend (API REST).

## 🎯 Funcionalidad General

👩‍⚕️ Gestión de pacientes y seudónimos por parte de la psicóloga.  
🎭 Asignación de identidad anónima al paciente.  
📝 Registro de encuestas post-sesión.  
📊 Visualización de semáforo de riesgo (🟢🟠🔴).  
🔐 Separación de roles y protección de datos sensibles.

## 🖥️ Frontend

Interfaz web para la interacción entre psicóloga y pacientes.

🚀 Funciones principales  
🎭 Selección de seudónimo mediante cards.  
📝 Encuestas de bienestar y diario personal.  
📊 Dashboard con indicadores visuales.  
👤 Vistas separadas según el rol.

🛠️ Tecnologías utilizadas  
🌐 HTML5.  
🎨 CSS3.  
⚙️ JavaScript (Vanilla JS).  
🔗 Consumo de API REST.

## ⚙️ Backend

API REST encargada de la lógica del sistema y la persistencia de datos.

⚙️ Funciones principales  
📦 Gestión de pacientes, seudónimos y encuestas.  
📈 Cálculo del estado de riesgo.  
🔐 Autenticación y control de acceso.

🛠️ Tecnologías utilizadas  
☕ Java 21.  
🌱 Spring Boot 3+.  
🔐 Spring Security.  
🗄️ MySQL.  
📦 Maven.
